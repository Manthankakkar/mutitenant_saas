Welcome to Multi-Tenant SaaS Demo

This demo shows a simple multi-tenant backend (Node.js + Express + Mongoose) and a static frontend using Bootstrap + Axios.

How to try
  1. Click Register to create a tenant + admin user.
  2. Then Login with tenant name, admin email and password.
  3. On dashboard you'll see tenant info and user list.
Project Files
  • Backend: Express, Mongoose, JWT auth
  • Frontend: Static HTML, Bootstrap, Axios
